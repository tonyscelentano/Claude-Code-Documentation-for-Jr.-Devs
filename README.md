LinkedIn is right - many Jr. Devs are shipping AI code, without understanding exactly how it works. I created a fairly simple thing to partially solve that.

It's just a Claude skill that inspects your app project and writes user-friendly documentation - everything that would normally go into ##hashtag#Comments in the code files, now go into a useful HTML documentation.

It covers ELI5, mermaid charts, instructions for manually modifying important code bits - basically documentation of your app, while AI builds it.
